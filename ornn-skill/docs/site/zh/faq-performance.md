# 性能常见问题

## 技能搜索有多快？

关键词搜索几乎即时。语义搜索（向量相似度）通常在 200-500ms 内响应，取决于技能库的规模。

## 执行时间限制是多少？

默认沙箱执行超时为 30 秒。可以在 SKILL.md frontmatter 中按技能调整。

## 技能包可以有多大？

最大 ZIP 上传大小为 **50 MB**。包内的单个文件应尽量保持较小。

## 执行是否支持并发扩展？

是的。chrono-sandbox 服务支持水平扩展。每次技能执行在隔离容器中运行。

## 试验场中的 LLM 延迟如何？

试验场响应通过 SSE 实时流式传输。首个 token 延迟取决于 LLM 提供商（通过 NyxID 网关），通常在 500ms 到 2s 之间。

## 有速率限制吗？

API 速率限制由 NyxID 管理。请在 NyxID 仪表板中查看当前计划的限制。
