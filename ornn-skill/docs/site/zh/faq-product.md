# 产品常见问题

## 什么是技能？

技能是打包的 AI 能力 — 由 SKILL.md 文件（包含元数据和文档）和可选的可执行脚本组成，可以被 AI Agent 执行。

## 支持哪些技能类型？

| 类型 | 描述 |
|------|------|
| **plain** | 仅提示词技能，无代码执行 |
| **tool-based** | 使用预定义工具/函数 |
| **runtime-based** | 在 Node.js 或 Python 中执行脚本 |
| **mixed** | 组合工具和运行时执行 |

## 如何创建技能？

三种方式：
1. **引导模式** — 逐步向导
2. **自由模式** — 上传预构建的 ZIP
3. **生成模式** — 描述需求，让 AI 构建

## 可以将技能设为私有吗？

可以。技能默认为私有。你可以随时在技能详情页切换公开和私有。

## 什么是试验场？

试验场是一个交互式环境，你可以在其中测试技能。它提供聊天界面，你可以询问关于技能的问题或使用真实环境变量触发执行。

## 认证如何工作？

Ornn 使用 **NyxID** 进行认证。通过 OAuth 使用 NyxID 账户登录。API 访问使用 NyxID API 密钥。
