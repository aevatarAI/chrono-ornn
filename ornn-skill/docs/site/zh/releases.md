# Ornn 版本发布

## v0.1.0-alpha — 首个 Alpha 版本（当前版本）

Ornn 技能平台的首个公开 Alpha 版本。本版本交付了技能创作、发现、测试和管理的核心用户功能。

### 核心功能

1. **登录与用户管理** — 与 NyxID 深度集成，支持 OAuth 登录、JWT 授权和基于角色的访问控制。

2. **三种技能创建模式**
   - **引导式** — 逐步向导，结构化创建技能
   - **自由式** — 上传预构建的技能包（ZIP）
   - **生成式** — 描述你的需求，让 AI 为你生成技能

3. **技能读取、更新与沙箱试验场** — 查看和更新技能包。交互式沙箱试验场支持对任意技能进行测试，试验场 Agent 通过将技能注入上下文的方式执行 Skill，涉及脚本执行时与 chrono-sandbox 集成。

4. **两种搜索模式**
   - **关键词搜索** — 传统的文本搜索
   - **语义搜索** — 基于向量嵌入，按含义查找技能

5. **管理员系统** — 管理员仪表盘，支持用户活动监控和平台级技能管理。

6. **多语言支持** — 完整的中文和英文本地化。

7. **主题颜色** — 深色模式和浅色模式。

8. **面向 AI Agent 的服务** — 提供 API 端点，供 AI Agent 进行技能搜索、技能抓取、技能上传与技能打造。

> 技术实现细节请参考**技术参考**部分。
