# Web 用户快速入门

## 1. 登录

通过 **NyxID** 登录：[https://nyx.chrono-ai.fun/](https://nyx.chrono-ai.fun/)。Ornn 依赖 NyxID 的用户系统与权限管理，NyxID 平台账号将对 Ornn 及其附属服务通用。

## 2. 浏览技能

点击导航栏中的**技能库**将自动跳转到公共技能库。点击右侧的**我的技能**切换到只有你自己可以访问的私有技能库。使用搜索栏进行关键词或语义搜索，按照名称或描述查找技能。

## 3. 创建一个新的技能

点击导航栏中的**构建**，选择创建方式：

| 方式 | 适用场景 |
|------|----------|
| **引导模式** | 首次创建者 — 逐步向导 |
| **自由模式** | 有经验的用户 — 上传预构建的 ZIP |
| **生成模式** | 描述你的需求 — AI 为你构建 |

### 引导模式

1. 点击**开始引导模式**
2. **基本信息** — 输入技能名称，选择分类，添加标签
3. **内容** — 使用 Markdown 编辑器编写 SKILL.md 正文（主要提示词内容），支持实时预览
4. **文件** — 添加任何支持文件（脚本、参考资料、资源）到技能包中
5. **预览** — 审查完整的技能包结构，然后点击**创建技能**

### 自由模式

1. 点击**开始自由模式**
2. 拖放一个 `.zip` 文件（或点击浏览）上传你预构建的技能包
3. 平台将自动验证包结构并提取元数据
4. 审查预览，然后点击**上传技能**

### 生成模式

1. 点击**开始生成模式**
2. 在输入框中描述你想让技能做什么
3. AI 将实时流式生成完整的技能
4. 在右侧预览面板中查看生成的 SKILL.md 和文件
5. 如需调整，通过后续消息进行优化，然后点击**保存技能**

## 4. 查看、编辑与测试你的技能

创建好的技能将存储在你自己的**私有技能库**。你可以选择将其公开与 Ornn 其他用户分享，或保留为私有技能（只有你可以用）。

现在点击你刚刚创建的技能，你会进入这个技能的详细信息界面：

- **左侧面板** — 整个技能包的内容。最上层的文件夹目录名字代表了你的技能名字。根目录下的 `SKILL.md` 是你整个技能的主要提示词内容，包括最上面的 YAML 格式 frontmatter 以及下方的提示词 Markdown 文本。你的 Skill 可能也会包含其他文件夹比如 `references/`、`assets/`、`scripts/`，这些文件夹中的内容与文件将成为辅助内容来支持你的技能。比如，一个调用 Gemini 图片生成帮你根据描述自动生成图片的技能，或许会调用代码/脚本才能完成这个工作，那么 `scripts/` 下面可能会存在一些脚本文件，以及 `SKILL.md` 的 frontmatter 中可能会出现一些环境变量、运行时与运行时依赖。

- 整个技能包中的内容你可以进行更改与删除，记得点击右上角的**保存**按钮来确认更新。点击保存时可能会有一个弹窗询问你是否要跳过 Validation，我们只建议高级用户选择跳过，因为这个 Validation 环节能够保证技能格式的统一性。

- **右侧面板** — 这个技能的描述、标签、技能作者以及一些其他元数据。下方你可以：
  - 点击**在试验场中尝试**来感受这个技能的作用
  - 点击**下载技能包**将技能下载到本地，配置给你的 AI Agent 使用
  - 如果你是这个技能的作者，也可以将技能设为公开/私有，或者删除它

## 5. 在试验场中体验

点击**在试验场中尝试**打开沙箱试验场。注意，如果这个技能需要任何环境变量依赖，记得在右上方的**环境变量**配置部分设置所需要的值。

现在你可以在左侧开始聊天，想象你正在使用一个装备了这个技能的 Agent，让它调用整个技能帮你执行一些相关操作。
