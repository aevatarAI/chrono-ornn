## What's New / 新增功能

### EN

- **NyxID Login** — Sign in with your NyxID account. Supports OAuth login and API key access.
- **Create Skills in 3 Ways** — Build skills step-by-step with the guided wizard, upload a pre-built package, or describe what you need and let AI generate it for you.
- **Skill Playground** — Test any skill interactively with an AI agent. Scripts run safely in a sandboxed environment.
- **Search the Skill Library** — Find skills by keyword or meaning. Semantic search understands what you're looking for even if you don't know the exact name.
- **Admin Dashboard** — Manage users, categories, and platform activity.
- **English & Chinese** — Full bilingual support. Switch languages anytime.
- **Dark & Light Mode** — Choose your preferred theme.
- **Agent API** — AI agents can search, download, upload, and generate skills programmatically.

### ZH

- **NyxID 登录** — 使用 NyxID 账户登录。支持 OAuth 登录和 API Key 访问。
- **三种方式创建技能** — 通过引导向导逐步创建，上传预构建的技能包，或描述你的需求让 AI 为你生成。
- **技能试验场** — 与 AI Agent 交互式测试任意技能。脚本在安全的沙箱环境中运行。
- **搜索技能库** — 通过关键词或语义搜索技能。语义搜索能理解你的意图，即使你不知道准确的技能名称。
- **管理员面板** — 管理用户、分类和平台活动。
- **中英文双语** — 完整的双语支持，随时切换语言。
- **深色与浅色模式** — 选择你喜欢的主题。
- **Agent API** — AI Agent 可以通过编程方式搜索、下载、上传和生成技能。
