## What's New / 新增功能

### EN

- **Skill Playground Chat** — Test any skill interactively with an AI-powered chat agent. The playground executes scripts in chrono-sandbox and streams responses in real time.
- **Login Session Fix** — Fixed an issue where your login session would be lost after refreshing the page or when the access token expired. Sessions now persist correctly across page reloads.
- **Version Display** — The current platform version is now shown in the footer so you always know which version you're running.

### ZH

- **技能试验场对话** — 通过 AI 驱动的对话 Agent 交互式测试任意技能。试验场在 chrono-sandbox 中执行脚本，并实时流式返回响应。
- **登录状态修复** — 修复了刷新页面或访问令牌过期后登录状态丢失的问题。现在会话状态能正确保持。
- **版本号显示** — 页面底部现在显示当前平台版本号，方便确认运行版本。
